## 2. FizzBuzz 問題のルール
1 から 100 までの数を標準出力に出力する。
- 3 の倍数の時は "Fizz" と出力する。
- 5 の倍数の時は "Buzz" と出力する。
- 3 と 5 の倍数の両方の時は "FizzBuzz" と出力する。
- その他の場合は、その数値をそのまま出力する。
